# Defining AI

AI can mean many different things in a project group. Use this page to make the definition explicit before choosing tools, data, or claims.

For this summit, define AI by what it does in the workflow:

- **Pattern recognition:** finding structure in images, text, spatial data, sensor records, or other observations.
- **Prediction or estimation:** generating a likely value, class, ranking, or scenario from available evidence.
- **Representation:** turning complex information into embeddings, summaries, labels, or features that can be reused.
- **Interaction:** helping people search, write, compare, debug, or reason through project material.
- **Automation:** carrying out repeatable steps that would otherwise require manual work.

## Questions to Answer

- What part of the project uses AI?
- What input does the AI system receive?
- What output does it produce?
- Who checks whether the output is useful or wrong?
- What should not be automated?
