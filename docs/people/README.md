# People Profiles

The public front page now links to existing learner profile files in the Innovation Summit 2026 repository instead of duplicating profile content here.

To add a person to the public front page:

1. Find their profile file in the learner folder:
   <https://github.com/CU-ESIIL/Innovation-Summit-2026/tree/main/docs/learners>
2. Add one Markdown list item to the People section of `docs/index.md`.

```markdown
- [Your Name](https://github.com/CU-ESIIL/Innovation-Summit-2026/blob/main/docs/learners/your-file-name.md)
```

Privacy rule: do not publicly expose both email and GitHub in the same visible profile block. Prefer one optional profile link rather than raw contact data.
