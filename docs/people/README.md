# People Profiles

Each person should have their own Markdown file in this folder.

To add yourself:

1. Copy `template.md`.
2. Rename the copy with your name, for example `alex-rivera.md`.
3. Edit only your own file.
4. Add the file to the People section of `docs/index.md` with a snippet include, such as:

```markdown
--8<-- "people/alex-rivera.md"
```

Privacy rule: do not publicly expose both email and GitHub in the same visible profile block. Prefer one optional profile link rather than raw contact data.
