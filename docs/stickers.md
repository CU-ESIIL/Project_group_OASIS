# Task Sticker Registry

Task stickers connect the public front page to the day-by-day directions. Each sticker has two explicit anchors:

- `edit-Dx-Y` on the public front page, where the group edits that part of the project record.
- `guide-Dx-Y` on the matching directions page, where the group reads the task instructions.

Click a sticker on the front page to jump to its directions. Click the same sticker in the directions to jump back to the front-page edit location.
